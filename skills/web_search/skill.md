# Skill: web_search

## 描述
搜索互联网获取最新信息，适用于需要实时数据、新闻、时事的问题。

## 触发条件
- 用户询问最新新闻、近期事件
- 用户询问实时数据（股价除外，股价用 stock_analysis）
- 问题包含"最新"、"现在"、"今天"、"搜索"等关键词
- 需要查找某个具体事实但不确定答案

## 不触发条件
- 问题可以直接用已有知识回答
- 涉及股票分析（交给 stock_analysis）
- 涉及文档问答（交给 document_qa）
- 涉及代码生成（交给 code_generation）

## 参数
- query: str  # 搜索关键词

## 返回
搜索结果摘要 + 来源
